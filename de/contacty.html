Contacty - Contact Microservice

Microservices implementieren per definition nur ein Feature. Dieser Microservice ermöglicht die Verschickung von Emails per Java Mail durch einen Rest mit JSON Schnittstelle. Dadurch kann man durch dieser Schnittstelle und Microservice Emails von jeglicher Programmiersprache verschickt werden. Dieses Microservice unterstützt SSL.

  • Diesen Microservice und Quellcode kann von den GitHub Repository heruntergeladen: https://github.com/daveajlee/contact-microservice
  • Benötigt wird mindestens Java 11 was ggf. hier heruntergeladen werden kann.
