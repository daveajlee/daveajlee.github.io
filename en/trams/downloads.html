Downloads

Before you can use TraMS, you will need to download TraMS onto your computer. This page provides instructions on how you can download TraMS onto your computer regardless of whether you are using Windows, Linux oder Mac OS.

How to download TraMS:

The latest version of TraMS (0.4.0). To download TraMS please choose your operating system from the list below and follow the installation instructions on the Documentation page.

  • Windows XP/Vista/7/8/10, Linux, Mac OS X and all other desktop operating systems

  • TraMS is not currently available for Android or iOS operating systems for mobile devices.
